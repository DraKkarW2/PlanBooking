# 🛫 PlanBooking - Gestion de Vols et Réservations en C++

PlanBooking est un projet C++ qui simule un système de gestion de compagnies aériennes.  
Il permet de gérer :

- Avions
- Vols
- Passagers
- Réservations (bookings)

Toutes les données sont stockées dans une base **MySQL**.  
Le projet a été développé rapidement pour un entretien technique, en utilisant une architecture simple **.h / .cpp** et une interface console.

---

## 🎯 Fonctionnalités

- Ajouter / Lister / Supprimer des avions, vols, passagers et réservations
- Vérification des clés étrangères (ex. pas de réservation sur un vol inexistant)
- Menu console simple et interactif 
- Utilisation de mysql et xamp pour l’accès à la base de données
